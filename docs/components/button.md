## API

| 属性 | 说明 | 类型   | 可选值       | 默认值 |
| ---- | ------ |------ | ------------ | ------ |
| icon | 图标名称  | string  | error / loading / settings / left / right / download / thumbs-up | / |
| iconPosition |  图标位置 | string | left / right | left |
| loading | 是否显示加载动画 | boolean | true / false | false |
